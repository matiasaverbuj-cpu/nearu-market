# NEARU Market — paquete para website

Este paquete contiene la identidad visual, las imágenes optimizadas y los textos base para construir el website de NEARU Market.

Dominio oficial: `nearu.market`.

## Estructura

- `logo/`: archivos SVG y PNG del logotipo, símbolo y favicon.
- `images/full/`: visualizaciones NEARU de los formatos Mini, Lobby, Container 20 FT y Container 40 FT en WebP y JPG, 1920 × 1080 px.
- `images/thumbs/`: miniaturas WebP, 960 × 540 px.
- `images/referencias-brasil/`: fotografías proporcionadas como referencias internacionales, optimizadas sin cambiar su contenido.
- `brand-guide.md`: reglas básicas de identidad y comunicación.
- `website-copy.md`: textos completos sugeridos para el sitio.
- `website-ai-prompt.md`: prompt listo para Claude/Cowork.
- `image-manifest.json`: relación de imágenes, formatos y textos alternativos.
- `email-structure.md`: estructura recomendada de correos.
- `name-validation.md`: resultado y alcance de la prevalidación pública del nombre.

## Uso recomendado

Usar WebP como formato principal y JPG como respaldo. El logo debe insertarse en SVG siempre que sea posible. No presentar las fotografías brasileñas como instalaciones ejecutadas por NEARU; publicarlas solamente como referencias internacionales y únicamente si se cuenta con autorización de uso.
